# 构建插件

:::tip

如果你是付费用户，那么可以跳过这一步。

:::

从 [Aiyatsbus 仓库](https://github.com/MCStarrySky/aiyatsbus) 中克隆完整项目在你的工作环境中。

```bash
git clone --depth=1 https://github.com/MCStarrySky/aiyatsbus
```

## 构建

```bash title="1. 构建"
gradlew build
```
