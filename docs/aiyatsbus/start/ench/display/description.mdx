# 描述

这部分内容涉及附魔的描述配置。

```yaml
display:
  description:
    general: "每击杀若干只怪物后，下一次攻击伤害增加"
    specific: "&7每击杀&a{击杀累计}&7生物后，下一次攻击伤害增加&a{伤害增加百分比}"
```

## 普通描述

即 `general`，一般不展示给玩家，玩家搜索附魔时会尝试匹配这个描述，而不是下面的展示描述。不建议使用变量和颜色。

## 展示描述

即 `specific`，一般在附魔大全或物品上展示给玩家。

可使用颜色符号，也可使用 [变量配置](/plugin/aiyatsbus/start/ench/variables/main) 中定义的变量，使用一个大括号括起变量名称。

:::tip

使用 **等级变量** 时，会自动加上单位。

:::

:::tip

可使用 `\n` 进行换行。

:::
