# 基本元数据

这部分内容涉及附魔的基本数据，如名称、最大等级、是否启用等。

```yaml
basic:
  enable: true
  disable_worlds: [ ]
  id: test
  name: 测试
  max_level: 3
```

## 节点介绍

| 节点            | 描述                       |
|-----------------|----------------------------|
| `enable`        | 是否启用                   |
| `disable_worlds`| 哪些世界里禁用           |
| `id`            | 附魔标识，仅允许小写字母及下划线 |
| `name`          | 附魔名称                    |
| `max_level`     | 附魔最大等级                |
| `rarity`        | 附魔品质 |

:::tip

关闭附魔后，物品上的附魔依然存在但不会生效，且该附魔不再能获取，也不会在附魔菜单中显示。

:::

## 附魔标识

附魔标识是辨别该附魔的唯一标识，不可与其他附魔的标识相同。附魔标识中仅允许包含小写字母及下划线。

:::tip

附魔配置的文件名无需与附魔标识保持一致，你可以给附魔配置起中文文件名。

:::
