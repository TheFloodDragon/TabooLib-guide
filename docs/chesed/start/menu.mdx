# 卸魔台

首先，使用命令打开卸魔台。若你在配置中配置了卸魔台方块，也可右键其来打开。

![](/img/Xnip2024-08-12_20-40-05.jpg)

打开后，我们可以看到装备放置处、辅助道具放置处和附魔产出区。我们需要将装备放进容器。

点击背包内的装备，即可上传，上传后即可看到产出附魔。

![](/img/Xnip2024-08-12_20-42-51.jpg)

附魔会按照等级排序。若你开启了品质排序，他一般按照更多附魔插件的显示模块中配置的品质顺序来排序。

将鼠标移动到附魔书上，即可查看对应的拆卸后等级与卸魔代价，点击即可拆卸附魔。

![](/img/Xnip2024-08-12_20-43-03.jpg)

默认用户一次解析的附魔数量有限制，可通过增加解析数量的道具与特权来增加解析数量。

:::tip

所有道具仅针对一次附魔拆卸，一次只可拆卸一个附魔。特权的加成一直拥有。

:::

![](/img/Xnip2024-08-12_20-43-13.jpg)

:::tip

黑名单中配置的附魔也无法被拆卸，会变成屏障附在末尾。

:::

![](/img/Xnip2024-08-12_20-44-34.jpg)

附魔列表上方的告示牌显示了本次拆卸附魔的信息，包括本次解析数量、使用的道具和特权。

![](/img/Xnip2024-08-12_20-50-17.jpg)

附魔列表一页可显示五个，可进行左右翻页。在附魔所在行的右面有羽毛和箭头。羽毛则为没有上/下页，箭头则表示有上/下页。

![](/img/Xnip2024-08-12_20-43-38.jpg)


点击附魔即可进行拆卸。

附魔祛除有成功率这个属性，若成功则尝试 **扣除费用**，若余额充足则视为拆卸成功，**会消耗全部已应用的道具**。若余额不足则无法拆卸，也不会消耗道具。

若失败，**也会消耗道具并扣除费用**，同时会返还原物品。

![](/img/Xnip2024-08-12_20-45-03.jpg)
